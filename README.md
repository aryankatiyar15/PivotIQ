# 🚀 PivotIQ — Startup Idea Validator

👉 **Live Demo:** https://pivot-iq.vercel.app  

**Clarity for your startup idea. Verdicts you can act on.**

## 🖥️ Best Experience

PivotIQ is currently optimized for desktop usage to ensure clarity in reading detailed validation reports and score breakdowns.

Mobile responsiveness is a planned improvement for future iterations.

---

## 🧠 Overview

PivotIQ is a **VC-style startup decision engine** that evaluates ideas through a structured conversational flow and delivers a clear **GO / NO-GO verdict**.

Instead of generic AI responses, PivotIQ simulates how a **skeptical investor** would analyze an idea — highlighting risks, weaknesses, and actionable next steps.

---

## 🎯 Problem

Most startup ideas fail not because of execution, but because of **weak assumptions**.

Existing tools:
- Give vague feedback  
- Feel like generic chatbot wrappers  
- Lack structured evaluation  

PivotIQ solves this by enforcing **clarity, structure, and critical thinking**.

---

## ⚙️ How It Works

1. **Guided Chat Input**
   - Problem
   - Target User
   - Solution
   - Competitors (optional)

2. **Evaluation Engine**
   - Problem Strength  
   - Market Potential  
   - Competition  
   - Monetization  
   - Execution Feasibility  

3. **Stress Test Layer**
   - Aggregates signals into a final score

4. **Verdict Generation**
   - GO / CONDITIONAL GO / NO-GO  
   - With reasoning and insights
